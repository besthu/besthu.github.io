Hive 多表FULL JOIN 解析

结论:

  • 多表FULL JOIN时,JOIN顺序影响最后展示结果
  • FULL JOIN是顺序执行,a FULL JOIN b FULL JOIN c等效于(a FULL JOIN b)的结果作为虚拟表与 c JOIN

三种查询对比如下:

first 

select a.salary,a.month,b.taxes,b.month,c.description,c.month
from a
full join  b
on a.month = b.month
full join c
on a.month = c.month

second

select a.salary,a.month,b.taxes,b.month,c.description,c.month
from b
full join c
on b.month = c.month
full join a
on b.month = a.month

third

select a.salary,a.month,b.taxes,b.month,c.description,c.month
from a
full join  b
on a.month = b.month
full join c
on coalesce(a.month, b.month) = c.month
